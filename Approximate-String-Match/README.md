# Approximate String Match

## Introduction

This is a subject project dedicated to using Edit Distance for Approximate String Match

## Methodology

### GED

Global edit distance

```
$ javac typosys.java

$ java typosys
```

### Damerau-Levenshtein Distance

```
$ javac transpos.java

$ java transpos
```

### Modified parameter

```
$ ./setup.sh
```
